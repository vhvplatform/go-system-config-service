# Flutter - Mobile Application

This directory will contain the Flutter mobile application for the System Config Service.

## Structure

The Flutter application will follow the recommended Flutter project structure:

```
flutter/
├── android/         # Android-specific files
├── ios/             # iOS-specific files
├── lib/
│   ├── main.dart    # Application entry point
│   ├── models/      # Data models
│   ├── screens/     # Screen widgets
│   ├── widgets/     # Reusable widgets
│   ├── services/    # API and business logic
│   ├── providers/   # State management
│   └── utils/       # Utility functions
├── test/            # Unit and widget tests
├── pubspec.yaml     # Dependencies and configuration
└── README.md        # This file
```

## Technology Stack (Planned)

- **Framework**: Flutter 3.x+
- **Language**: Dart 3.x+

- **Local Storage**: Hive / Shared Preferences
- **Navigation**: GoRouter

- Secret management interface
- Push notifications for configuration changes
- Offline support with sync
- Multi-environment switching
- Biometric authentication

## Platform Support

- Android 6.0+ (API level 23+)
- iOS 12.0+
